The following describes map options. Example: 

  $default_map = array(
    'options' => array(
      'displayProjection' => '4326',
      'maxResolution' => 156543.0339,
      'maxExtent' => array(
        'left' => -20037508.34,
        'bottom' => -20037508.34,
        'right' => 20037508.34,
        'top' => 20037508.34,
      ),
      'fractionalZoom' => ??
    ),
  );
  • displayProjection
    Values:
    string (number)
    Description:
    The display projection to use.
    Example:
    '4326'
  • maxResolution
    Values:
    float
    Description:
    The max resolution.
    Example:
    156543.0339
  • maxExtent
    Values:
    array
    Description:
    An array of maximum extent that will be shown. This should include the values: left, right, bottom, top.
    Example:
    array(
  'left' => -20037508.34,
  'bottom' => -20037508.34,
  'right' => 20037508.34,
  'top' => 20037508.34,
),
